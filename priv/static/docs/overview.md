# Overview

Nindo is a RSS reader and blogging platform designed to be easy and minimal.

![https://raw.githubusercontent.com/RobinBoers/nindo-phx/master/priv/static/images/post.png]

## Features

- Markdown support
- Comments
- Dark mode
- Readable
- RSS feeds
- REST API
- Responsive and quick
